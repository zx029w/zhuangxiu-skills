# 装修公司避雷指南

## 一、使用说明

本Skill提供装修公司筛选的专业技巧，帮助您做出更明智的选择。

**重要提示：**
- 本Skill仅汇总公开信息，不构成任何决策建议
- 信息准确性需您自行核实
- 建议在专业人士指导下决策
- 选择装修公司前，请务必实地考察、多方对比、仔细审核合同

## 二、核心功能

### 功能1：筛选技巧指南

从知识库提取"如何选择装修公司"的专业方法。
详见 `references/选公司与筛选技巧_一核心原则.md`
详见 `references/选公司与筛选技巧_二找公司方法.md`
详见 `references/选公司与筛选技巧_三淘汰法与进阶.md`
详见 `references/选公司与筛选技巧_四真相与方向.md`

核心要点：
- 先看人品，再看价格
- 不量房出预算 → 直接淘汰
- 套餐公司 → 谨慎或直接绕开
- 平台找公司 → 多了一层贩子
- 签合同前必须看3点：环保态度、咨询行为、预算详单程度

### 功能2：风险识别指南

列出需要警惕的常见问题（基于知识库内容）。
详见 `references/淘汰红线项清单_一假忠告与信息差.md`
详见 `references/淘汰红线项清单_二36法则与红线.md`
详见 `references/淘汰红线项清单_三报价陷阱与利润.md`

**5条绝对不能碰的红线：**
1. 别贪便宜找散工
2. 别迷信"独立设计+施工队"省钱高端
3. 选公司别只看名气和规模
4. 别靠装修平台找公司
5. 慎选按平方计价的套餐装修

**装修公司6不找：**
1. 没有固定地点的公司不找
2. 套餐不找（土豪除外）
3. 平台不找（增加一个贩子）
4. 非详单式报价不找
5. 转包的最好不要找
6. 装修不思考的机构不要找

**签合同前3点判断良心机构：**
1. 环保态度：是否替业主着想而非替人民币着想
2. 咨询行为：是否围绕户型讲解而非只推活动礼品
3. 预算是否详单式：能否看懂每一项

### 功能3：指定公司查询

当用户输入**具体的装修公司名称**时，按以下流程处理：

Step 1: 从知识库提取筛选技巧 → **始终输出**

Step 2: 搜索公开平台的可查信息 → 汇总后输出

**【重要规则 - 必须遵守】**

对于**通用咨询类**问题（如"哪个装修公司好？"、"装修公司排名"、"怎么判断好坏"等，无具体公司名时）：
- 优先从知识库输出**筛选技巧**（详见`references/选公司与筛选技巧.md`）
- 输出**避坑红线和常见陷阱**（详见`references/淘汰红线项清单.md`）
- 可输出优质机构的通用参考标准供对比
- **不需要也不应该**调用公开信息搜索功能

对于**指定公司名称的查询**（如"XX装饰怎么样"、"帮我查一下XX公司"）：
- 仅搜索并汇总该公司的公开可查信息
- 不输出该公司的任何正面宣传、好评、案例等信息
- 所有信息均标注来源，供用户自行判断
