## 三、怎么找装修公司（178篇干货精华）【旧内容保留作为补充】

### 核心原则

**先看人品，再看价格。** 装修不是买白菜，越便宜越要警惕。合理的价格+透明的报价才是正道。

**好装修公司的标准：**
- 用人品做装修，先做人后做事
- 不做套餐，只做业主必须做和愿意做的
- 不会中途产生额外费用（故意偷项漏项业主可拒绝付款）
- 所有装修猫腻都明明白白告诉你
- 统一签订正式合同（建委2007版）
- 水电项目户户估价
- 每户配发《工程手册》，材料/阶段验收签字制
- 根据实际套内面积计算价格，不以建筑面积笼统算
- 免费量房做预算，不提前收任何费用

### 找装修公司的渠道

1. **口碑推荐** — 最可靠的方式，客户靠口碑而来不是靠广告抢来的
2. **实地考察** — 去正在施工的工地看看真实情况
3. **看预算单** — 是否为详单式报价（以你家区域为块，每块详细列项目）
4. **看人** — 对接负责人的人品、责任心和专业度

### 怎么判断一个装修公司好坏

**看咨询过程：**
- 围绕你家户型讲解难点 → 正常机构 ✅
- 只推活动、礼品、优惠 → 小心 ⚠️
- 进门满眼礼品金蛋条幅 → 别进 ❌

**看环保态度：**
- 明知有小孩还推荐木地板/壁布 → 替人民币想 ❌
- 主动提醒环保问题 → 替你想 ✅

**看预算方式：**
- 量房免费、初步方案免费、预算免费 → 合理 ✅
- 不量房就出预算 → 挨宰 ❌
- 以建筑面积笼统计算 → 不专业 ❌

**看报价形式：**
- 详单式（每区域每项目明细清楚）→ 专业 ✅
- 定额式/复杂看不懂的格式 → 有猫腻 ❌

### 自装 vs 找装修公司

| | 自己装 | 找装修公司 |
|---|---|---|
| 时间 | 全程操心 | 省时省力 |
| 材料 | 自己把控质量 | 可能以次充好（全包时） |
| 风险 | 出问题自己扛 | 有人兜底售后 |
| 价格 | 可能更便宜但返工更多 | 明码标价 |
| 建议 | 只有懂行且有时间的才自装 | 大部分人还是找正规公司 |

### 半包 vs 全包 vs 套餐

**半包** — 公司提供人工+辅料，主材业主自己买
- 优点：把握主材质量、避免浪费
- 缺点：主材风格颜色自己把握、购买超支无折扣

**全包** — 公司全权委托，业主只买灯具开关家具家电
- 优点：省时省力、风格统一、可能节省一部分费用
- 缺点：无法把握材料好坏和价格、担心以次充好

**套餐** — 按平米计价的全包模式
- ⚠️ **强烈不推荐！** 低价切入后期疯狂增项
- 套餐表象唯一优点就是低价，实际装下来比正常贵一倍以上

### 找多少家公司咨询合适？

建议找3-5家不同类型的公司对比：
- 1家本地口碑好的老牌公司
- 1-2家中型正规公司
- 了解但不急于签套餐公司（用来对比价格陷阱）

### 装修前必做的功课

1. 了解基本行情（大概多少钱一平）
2. 确定自己的装修方式和预算范围
3. 准备好户型图
4. 列好自己的需求和优先级
5. 了解主要材料的品牌和大致价格

### 避免踩坑的关键心态

- 不要有从众心，有主见者得天下
- 装修噱头分不清，难道等坑了再骂？
- 不思考黄绿蓝惊醒不了麻木的人
- 不懂行+不识人+烂公司 = 三大灾难
- 装修差不多就选价格低的？大错特错！
