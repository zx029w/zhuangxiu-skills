# 瓷砖与墙面问题补救

> 来源：西安装修课堂知识库「瓷砖空鼓怎么办？砸了重做？」「已经入住，地面砖出现空鼓，灌浆处理即可」「乳胶漆别刷错！墙面反复开裂？」「经验|吊顶开裂不发愁」等文章提炼。

## 一、瓷砖空鼓问题

### 1. 瓷砖空鼓的原因
- 工人铺贴问题，水泥砂浆没涂抹均匀（多见四角局部空鼓）
- 踩踏过早，瓷砖没干透就上去踩
- 基层没处理好

### 2. 空鼓的判断方法
- 用空鼓锤或小锤子敲击瓷砖，听声音
- 沉闷声 → 正常
- 清脆声 → 空鼓

### 3. 不同位置的空鼓处理

**地面瓷砖空鼓：**
- 经常踩踏的地方（过道、客厅）必须处理，否则会扩大
- 短时间内更换掉
- 入住后才发现？→ 灌浆处理

**墙面瓷砖空鼓：**
- 单片空鼓超总面积10% → 建议砸掉重贴
- 少量空鼓在非关键位置 → 可以观察不做处理

### 4. 灌浆处理方法（已入住补救）
适用：地面瓷砖空鼓，已经入住不方便大动干戈

步骤：
1. 用壁纸刀把空鼓砖的缝隙清理干净
2. 清理到可以感觉到瓷砖下方有空隙
3. 纯水泥加水搅拌，必须要非常稀（太稠进不了缝隙）
4. 慢慢倒入清理好的瓷砖缝隙
5. 让水泥浆自然流入空鼓部位
6. 干透后清理表面多余水泥

## 二、墙面开裂问题

### 1. 墙面开裂的根源（根不在漆，在基层）
10大基层细节：

**① 素灰没铲除** → 新房毛坯房的素灰必须铲除到底，露出原始基层
**② 大白没铲干净** → 开发商刮的大白必须铲干净，否则后期开裂脱皮
**③ 松散沙灰层** → 掉渣掉粉的沙灰层必须铲除到底
**④ 老房水泥层** → 检查空鼓，有空鼓敲掉；没有且坚硬则无需铲到红砖层
**⑤ 非承重墙松散** → 用钥匙划一下，掉粉脱落要铲到底
**⑥ 墙面空鼓** → 必须敲掉铲除，重新粉刷石膏找平（不要用水泥砂浆）
**⑦ 墙面挂网** → 旧房铲墙99%需要全房挂网；新房大面积找平也要挂网
**⑧ 不同材质墙体缝隙** → 最低标准：一道牛皮纸+两道网
**⑨ 石膏板缝隙** → 同样一道牛皮纸两道网
**⑩ 石膏砂浆抹灰层** → 必须全房敲空鼓

### 2. 墙面开裂的补救
- 小裂纹：铲掉腻子，贴绷带，重新批腻子刷漆
- 大裂缝：铲到基层，重新做石膏找平，挂网，批腻子，刷漆
- 反复开裂：说明基层问题没解决，需要彻底处理

### 3. 墙面返碱/发霉的补救
- 返碱原因：卫生间防水没做好，潮气渗透
- 补救：先解决防水问题，再铲掉发霉墙面，重新做
- 预防：卫生间防水至少做到1.8米（建议2.4米到顶）

## 三、吊顶开裂问题

### 1. 吊顶开裂原因
- 房屋存在沉降
- 吊顶材料在空气中变形
- 开空调温差大
- 石膏板接缝处理不到位

### 2. 吊顶开裂修复步骤
1. 把腻子铲掉
2. 石膏板划倒V形
3. 用石膏嵌满
4. 贴上绷带
5. 批腻子
6. 做涂料

### 3. 预防吊顶开裂（装修时）
- 顶面沿墙一圈阴角预留伸缩缝
- 石膏板转角用整板裁切（L形）
- 石膏板与墙面留缝3-5mm

## 四、水槽龙头与窗户冲突
- 利用水槽三个预留孔位，选择合适位置安装龙头
- 或更换折叠龙头解决

## 五、卫生间外墙发霉脱皮
- 原因：卫生间防水不到位，潮气渗透
- 处理：
  1. 先解决防水问题（重做或修补）
  2. 铲掉发霉脱皮的墙面
  3. 重新批腻子刷漆
  4. 如果防水已做好还发霉，检查是否水管渗漏
- 预防：防水做高（到顶最好），卫生间保持通风
