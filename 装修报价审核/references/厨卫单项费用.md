# 厨卫单项费用

> 来源：西安装修课堂·陈工 | 精选11篇文章提炼

## 一、一间卫生间装修花多钱

以6平米干湿分离卫生间为例（毛墙毛地开始）：

| 项目 | 费用 |
|------|------|
| 水电包工包料 | 1000元 |
| 水泥沙子、包管道、做壁龛 | 1000元 |
| 防水（18平米×60元） | 1080元 |
| 防水保护层 | 200元 |
| 墙砖（300×600，123块×15元） | 1845元 |
| 地砖（300×300，60块×10元） | 600元 |
| 铺贴瓷砖（37平米×50元） | 1850元 |
| 瓷砖倒角（15米×20元） | 300元 |
| 铺砖拉毛费用 | 200元 |
| 铝扣板吊顶（6平米×170元） | 1020元 |
| 湿区金属门 | 1000元 |
| 干区浴柜（含台下盆、龙头、镜柜） | 2000元 |
| 湿区浴霸 | 2000元 |
| 干区镜前灯 | 100元 |
| 地漏（潜水艇2个） | 160元 |
| 马桶 | 1000元 |
| 花洒 | 1000元 |
| 毛巾架五金 | 500元 |
| **合计** | **约16855元** |

**一间普通卫生间装修下来将近两万元**，而且用的东西都不敢用奢华的。一个浴柜七八千都有，座便器四五千太多，毛巾架五金件不锈钢的一两千都是正常的，以上只是满足功能性。

## 二、一个厨房装修要花多钱

厨房装修费用参考（毛墙毛地开始）：
- 水电改造：约1000元
- 防水（有地漏才做，无地漏坚决不做）：约500-1000元
- 墙砖地砖：根据面积和瓷砖价格
- 铺贴费用（含倒角、拉毛）：根据面积
- 铝扣板吊顶：约800-1500元
- 橱柜（地柜+吊柜）：3000-8000元（看材质和长度）
- 水槽套装：500-2000元
- 烟机灶具：2000-5000元
- 厨房门：800-2000元

**一个普通厨房装修下来约1.5-3万元**，取决于材料档次。

## 三、厨房有地漏做防水，无地漏坚决不做

**这是实践总结的经验，也是该花的必须花、不该花的坚决不花的原因所在。**

- 有地漏的厨房：需要做防水，因为地漏可以做排水
- 无地漏的厨房：做防水毫无意义，水无处排，做了也是白做

**装修机构故意给你做厨房防水**有两种可能：要不不懂工艺，要不故意多赚钱。不懂工艺的，必定会给卫生间少做防水，因为肯定不懂卫生间防水是防潮气的标准。

## 四、卫生间防水的正确做法

**卫生间防水一定是防潮的，不是暂时不漏水。**

1. **湿区防水做到吊顶位置**，不是只做1.8米
2. **干区没有地漏不需要做防水**
3. **防水必须做两遍**，用彩色防水涂料（蓝色），刷到什么程度一眼能看出来
4. **做完防水的墙面需要拉毛处理**才能贴砖
5. **防水保护层**不能省

**低价切的做法：** 淋浴处1.8米高，其他地方上翻30公分。进场再告诉你不做高的危害，让你加钱。

## 五、厨卫吊顶大有讲究

### 铝扣板吊顶
- 厨卫最常用的吊顶方式
- 价格：50-170元/平米不等
- 不要追求所谓大品牌，厂家直接做的就行
- 注意：吊顶里头的隔音棉一定要做吊顶的一块做，不要分开

### 厨卫有大梁怎么处理
- 大梁位置需要特殊处理吊顶
- 不能直接用铝扣板压过去
- 需要结合梁高做异形处理

### 吊顶和隔音棉
- 卫生间竖管包红砖不需要隔音棉
- 吊顶里头裸露的管道需要做隔音棉
- 隔音棉一定要做吊顶的一块做，不要分开找人做

## 六、包管道和包烟道

**竖管包红砖：** 水泥砂浆砌120红砖墙，不需要隔音棉
**吊顶里头管道：** 需要做隔音棉
**烟道处理：** 必须红砖包了才贴瓷砖

**经常被故意漏项的地方：**
1. 包管道费用不在预算里
2. 烟道处理费用不在预算里
3. 管道超过40公分加收100元/根的条款

## 七、过门石和倒角

### 过门石
- 厨房没有地漏：公共区域大砖能通铺尽量通铺，除非无法对缝才用过门石
- 厨房有地漏：必须采用过门石，因为需要做防水
- 每个区域有过门石的地方都需要铺贴费用

### 倒角
- 建议倒角，比压条美观
- 注意倒角米数：一个厨房门洞倒角就五六米，加上烟道管道窗台，再小的空间低于十米都得注意
- 倒角是两块瓷砖，20元/米可能变成40元/米

## 八、插座预留

**装修插座留多少？这些插座预留位置一定要记住：**
- 厨房：油烟机、冰箱、微波炉、电饭煲、水槽下方（净水器/垃圾处理器）、台面备用
- 卫生间：镜前灯、浴霸、智能马桶、吹风机
- 客厅：电视、空调、沙发两侧、茶几下方
- 卧室：床头两侧、空调、书桌
- 阳台：洗衣机、热水器

**厨房水槽水电怎么预留：**
- 冷热水管预留高度：45-50cm
- 净水器电源：水槽下方
- 垃圾处理器电源：水槽下方
- 水槽下方留够空间和插座

## 九、同样一个房子装修，单价高的总价却低

这是一个迷惑性很强的现象。单价高但总价低的原因：
1. **面积算得准：** 没有故意少算，后期不加钱
2. **项目算得全：** 没有漏项，后期不增项
3. **做法算得细：** 备注清楚，没有模糊条款
4. **材料算得明：** 品牌型号写清，不存在升级加钱

**单价低总价高的原因：**
1. 面积故意少算 → 后期加钱
2. 项目故意漏项 → 后期增项
3. 做法故意模糊 → 后期坐地起价
4. 材料只写品牌不写型号 → 后期升级加钱

**结论：不要只看单价，也不要只看总价，要看项目是否齐全、做法是否透明。**
