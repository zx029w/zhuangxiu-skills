# 柜门五金把手类

> 本文件收录全屋定制柜门材质选择、把手设计、五金配置等核心知识，来源：从学到装｜装修课堂知识库。

## 一、三种主流柜门材质对比

### 1. 吸塑柜门

**基材：** 高密度板，表面高温吸膜
**规格：** 1220mm × 2440mm

**优点：**
- 可以做平板也可以做任意造型（电脑雕刻）
- 颜色上千种，几乎满足所有风格需求
- 肤感柔光系列手感顺滑、视觉干净
- 可以做整体百叶门（天然气表、鞋柜百叶），材质统一美观
- 覆膜工艺，正面看不到封边条

**缺点：**
- 高光系列不耐划痕，短时间就有划痕
- 基材是高密度板，环保性一般

**适用：** 美式造型柜门、百叶门、各种纯色柜门

### 2. 多层柜门

**基材：** 实木多层板
**规格：** 1220mm × 2440mm

**优点：**
- 木纹纹路比吸塑更逼真自然
- 耐磕碰性能好
- 适合做木色系列柜门
- 可做浴柜柜门（只要封边不被破坏、不大量泡水）

**缺点：**
- 纯色系列很少，多为木纹色
- 无法做造型（只能平板）
- 不宜过高，太高容易变形
- 有封边条，正面可见

**适用：** 厨房地柜木色柜门、衣柜木色柜门、浴柜

### 3. PET柜门

**基材：** 高密度板，表面PET塑料覆膜
**规格：** 可做到2.7米（但不推荐做太高）

**优点：**
- 环保性好（矿泉水瓶子材质就是PET）
- 柔光肤感效果比吸塑略胜一筹
- 耐磕碰性能好
- 颜色纯度高

**缺点：**
- 只能做平板，无法做造型
- 有封边条（同色封边不仔细看看不到）
- 高光系列不耐划痕
- 百叶门不如吸塑美观（需框架镶嵌铝合金百叶）

**适用：** 现代极简风纯色柜门、高柜柜门

### 三种材质选择建议

| 需求 | 推荐材质 |
|------|---------|
| 纯色平板极简风 | PET（肤感）或吸塑（肤感） |
| 造型柜门（美式等） | 吸塑 |
| 木色柜门 | 多层 |
| 百叶门 | 吸塑 |
| 浴柜柜门 | 多层 |
| 厨房地柜 | 多层（耐磕碰） |
| 厨房吊柜 | 吸塑或PET（纯色） |

## 二、四种不推荐的柜门

### 1. 推拉门（坚决不推荐）

**缺点：**
- 轨道占用衣柜内部空间，衣柜缩小
- 轨道积灰尘，难清理
- 家有小孩容易夹手
- 轨道变形后柜门无法关闭严实
- 颜值低

### 2. 亮光柜门（不推荐）

**缺点：**
- 亮光自带廉价感
- 张扬夸张，与内敛审美背道而驰
- 不耐划痕
- 晶钢柜门彻底翻车

### 3. 一门到顶（不推荐）

**缺点：**
- 必须工艺满足（墙面横平竖直），否则缝隙不均匀
- 柜门变形概率高（不论用什么材料）
- 上碰吊顶、下碰脚
- 拉直器是智商税，装了照样变形
- 衣柜最高处无法挂衣服，只能放杂物被褥
- 断开设计（分段式）更好用、颜值也高

### 4. 玻璃柜门（慎选）

**缺点：**
- 过日子不是展示，入住后会很乱
- 现实和梦想有差距
- 容易翻车

## 三、把手选择指南

### 1. 三种该淘汰的把手

**视觉厚重的拉手：**
- 像豪宅大门一样突兀
- 再好的衣柜都能毁掉
- 廉文感满满

**通柜门拉手（土豪金通条）：**
- 比厚重拉手更显得厚重和廉价
- 高调突兀，弱化存在感的目的完全没达到
- 上下还容易翻车

**一门一拉手（明装突兀型）：**
- 小但非常突兀
- 多此一举
- 把同色平板衣柜的价值毁于一旦

### 2. 正确的把手方案

**公共区域（客厅、餐厅、阳台）：** 尽量免拉手
- 拉槽设计（免费）— 首选
- 柜门下挂0.5-1公分（免费）— 吊柜或下方有空间的柜门
- 反弹器（普通品牌免费）— 实在没办法时配合拉槽使用

**卧室衣柜：** 极简拉手
- 一定要极简，弱化存在感
- 两个柜门共用一个把手（不需要一门一个）
- 明装把手也要选视觉轻巧的

**橱柜：** 免拉手拉槽设计
- 视觉更极简干净
- 比金属条免拉手更实惠（金属条要收费且需裁割柜体）

### 3. 无把手设计实操

**鞋柜无把手做法：**
- 下方柜门采用拉槽设计
- 上方柜门下降5毫米
- 不用一个反弹器
- 有抽屉时，抽屉和柜门共用一个拉槽
- 所有木板保证机器封边，造价在投影计价内

**餐边柜无把手做法：**
- 下方拉槽设计 + 上方柜门下放
- 不用反弹器
- 玻璃柜门自带拉手

**满墙柜无把手做法：**
- 上柜门下放
- 下方抽屉板下放后从下方拉开
- 玻璃柜门自带拉手

### 4. 免拉手收费 vs 免费

| 做法 | 是否收费 |
|------|---------|
| 普通拉槽 | 免费 |
| 柜门下挂 | 免费 |
| 反弹器（普通品牌） | 免费 |
| 斜切45度 | 收费 |
| 金属条免拉手 | 收费（还需裁割柜体） |
| 铣槽拉手 | 收费 |
