# 定制与安装翻车急救

> 来源：西安装修课堂知识库「78篇装修翻车警示」、「橱柜定制翻车实录」、「衣柜定制翻车教训」、「18个卫生间翻车事故」等文章提炼。

## 一、全屋定制翻车

### 1. 尺寸测量翻车
- 只量了大概尺寸，没考虑门套、踢脚线、墙体不直
- 补救：
  - 缝隙小 → 用收边条/玻璃胶填补
  - 缝隙大 → 定制亚克力板或加做封板
  - 切割板材（橱柜/台面可现场切割）

### 2. 柜门变形/色差
- 颗粒板柜体受潮变形
- 补救：更换为多层实木板
- 色差：能接受就不处理，不能接受联系商家更换

### 3. 定制开孔露截面（颗粒板）
- 补救：网购自沾封边条，颜色相近更好

### 4. 柜子与墙面颜色不统一
- 补救①：按柜子侧面尺寸定制全身镜
- 补救②：软木板/黑板墙遮挡

### 5. 衣柜层板做少了
- 补救：网购层板架实现灵活收纳

### 6. 鞋柜定制翻车
- 鞋柜太深或太浅 → 太深做活动层板，太浅做斜插层板
- 吊顶和鞋柜不协调 → 加做封板

### 7. 餐边柜定制翻车
- 尺寸不对放不进 → 切割或重做
- 后悔莫及 → 联系商家修改

### 8. 一门到顶变形
- 2.8米以上柜门必变形
- 补救：加装拉直器（效果有限）/更换为分体柜门

## 二、安装翻车

### 1. 开关插座安装问题
- 插座安歪了 → 拆除重装（乳胶漆墙面好处理，瓷砖墙面麻烦）
- 插座缝隙大 → 乳胶漆墙面油漆工返场修补
- 瓷砖墙面插座缝隙 → 面板装饰盖/遮丑神器

### 2. 灯具安装问题
- 灯偏了不在餐桌正上方 → 灯具移位器（几十元）
- 灯具太高或太低 → 调节吊线长度

### 3. 花洒安装翻车
- 出水口不齐/突出 → 万能曲角+加大装饰盖
- 顶喷与吊顶冲突 → 铝扣板可拆，让花洒穿过吊顶

### 4. 马桶安装问题
- 背后有缝隙 → 移位器/密封条/亚克力板
- 法兰圈没装好返臭 → 拆了重装法兰圈

### 5. 门安装翻车
- 门装反/装歪 → 拆了重装
- 门套太厚突出墙面 → 定制石膏圆弧角弱化
- 金属门无法装门吸 → 硅胶防撞门吸

### 6. 橱柜安装问题
- 烟机与吊柜缝隙 → 烟机伸缩架
- 吊柜与吊顶冲突 → 加做封板

### 7. 踢脚线问题
- 踢脚线没装好拖地磕墙 → 重新安装或更换
- 踢脚线缝隙大 → 玻璃胶填补

### 8. 电器安装翻车
- 冰箱尺寸不对放不进 → 退货换尺寸/拆柜门
- 洗碗机装不进 → 拆柜体或换机器
- 洗衣机地漏不下水 → 疏通或改管

## 三、卫生间翻车急救

### 1. 卫生间坡度做小了
- 补救：做挡水条+淋浴区单独处理
- ⚠️ 严重的话需要砸掉重做地砖

### 2. 卫生间外墙发霉脱皮
- 原因：防水不到位
- 处理：先解决防水，再铲墙重做

### 3. 马桶后面缝隙
- 见通用补救方法

### 4. 浴柜安装缝隙
- 见通用补救方法

### 5. 卫生间没留镜前灯/插座
- 补救：充电式镜前灯/带USB的镜柜

### 6. 卫生间漏水
- 见水电与渗漏问题急救

## 四、常见翻车案例警示

### 1. 厨房翻车
- 动线翻车（洗切炒顺序不对）→ 只能适应，无法改变
- 厨房铺地毯 → 油污地狱，撤掉
- 厨房吊柜与吊顶翻车 → 加封板或改吊柜高度

### 2. 客厅翻车
- 电视墙翻车 → 太复杂不好看，刷白或贴壁纸覆盖
- 客厅灯翻车 → 换灯
- 沙发插座被挡住 → 嵌入式插座/沙发充电宝

### 3. 卧室翻车
- 床头插座高度不对 → 反装插座/多功能床头柜
- 双控没做 → 无线遥控开关
- 衣柜翻车 → 联系定制修改

### 4. 阳台翻车
- 阳台定制柜翻车 → 改柜子
- 阳台漏水 → 排查找漏点维修
- 阳台下水管处理不当 → 重新包管

### 5. 飘窗翻车
- 飘窗靠窗装窗帘漏光 → 10种补救方法
  - 加窗帘盒
  - 换百叶帘
  - 纱帘靠窗+布帘满墙
