# 预算陷阱避坑

> 来源：西安装修课堂·陈工 | 精选12篇文章提炼

## 一、21个报价单隐藏项

装修报价里的套路太多，很多商家把隐藏费用藏在报价单里，业主签完字才发现后期不断加钱。

### 水电部分（7个坑）
1. **开槽费、封槽费单独加收** —— 很多业主以为含在单价里
2. **底盒、弯头、接头等辅材后期计价** —— 一定问清是否包含
3. **水电点位超出约定数量加钱** —— 套餐公司惯用伎俩
4. **水管走顶保温棉单独计费**
5. **开槽位置单独做防水处理** —— 没用的表演项目单独计费
6. **按米计价不估价不谈实际出入** —— 总价靠运气
7. **水电签进合同** —— 断了自己后路，无法退出

### 墙面施工（3个坑）
1. **粉刷石膏找平是否包含** —— 几乎每家墙面都需要找平
2. **挂网、网格布、牛皮纸是否包含** —— 旧房铲除后必须全房挂网
3. **墙面特殊处理** —— 返碱发霉处理、阴阳角条、阳角弧度处理是否单独收费

### 瓦工部分（7个坑）
1. **瓷砖背胶、瓷砖胶、十字卡、找平器** —— 是否包含还是自购
2. **墙面防水拉毛、墙面误差找平** —— 是否单独收费
3. **大规格瓷砖、异形砖、斜铺、拼花** —— 是否单独加钱
4. **瓷砖切割、倒角、海棠角、开孔** —— 是否都单独预算
5. **薄贴找平费用** —— 地面水泥砂浆铺贴不需要找平，是否有重复收费
6. **勾缝工费** —— 不做美缝采用普通勾缝，是否有单独工费
7. **防水高度故意做低** —— 后期再增项；防水只刷一遍，后期加钱

### 拆改与垃圾清运（4个坑）
1. **拆除项目和拆除结果要列清** —— 防止进场低开高走
2. **垃圾运至物业指定点费用是否包含** —— 外运如何收费
3. **老房拆改到什么层** —— 沙灰层还是红砖层，成本不同
4. **垃圾下楼方式** —— 云梯车还是人工运输，费用不同

## 二、装修为什么会超预算

**预算"不靠谱"，合同形同"废纸"。** 超预算的核心原因：
1. **低价切入：** 故意漏项让合同价低，进场后不断加钱
2. **模糊条款：** 备注里写"超过X加钱"，但从不提"低于X减钱"
3. **拆项计算：** 一个项目拆成几个，看起来单价低，加起来总价高
4. **按实际发生：** 水电、防水、开关安装等不估价，后期坐地起价
5. **不量房出预算：** 没有准确工程量，后期面积增加就加钱

## 三、预算里的智商税

1. **地固：** 纯智商税。铺瓷砖用水泥砂浆，防什么尘？固什么沙？
2. **开工黄墙绿地：** 面子工程。墙固是批腻子前才刷的，开工先刷等于白刷。
3. **管理费：** 一般5%-10%，是装修公司利润的一部分。不收管理费其实揉到其他单价里了。
4. **成品保护费：** 避免原有成品受损加以保护的费用，好公司应该包含在管理费里。
5. **材料搬运费：** 装修中所有乙方购买材料的搬运费用，应该包含在单价里。

## 四、装修预算只有单价坑你没商量

**只有单价没有总价的项目，都是为了让合同价看起来低。**

量房的意义在于：专业人士上门，应该知道每家的大概情况。既然量房了，所有可以精确测量的项目都应该一次性算好，而不是只给单价后期做多少算多少。

如果装修公司说"这个无法确定，只能按实际发生"，你要判断：
- 水电确实无法提前确定 → 可以理解，但必须估价谈实际出入
- 防水面积 → 卫生间就那么大，完全可以算出来
- 开关插座数量 → 做完水电就知道，可以包死
- 瓷砖倒角米数 → 量房后可以计算
- 包管道长度 → 完全可以测量

## 五、低价切入与套餐陷阱

### 套餐公司的套路
套餐公司号称"全包带回家，包含全房家具、家电"，实际上：
1. **定金套住客户：** 定金不可退，一万或两万
2. **设计师其实是业务员：** 上来一番洗脑，聊优惠不聊装修
3. **主材不怎么样：** 品牌限制，选择有限，都是专供猫腻
4. **个性化全无：** 窗套、阳台门套、垭口、地面拼花、吊顶造型都不包或明显不全
5. **报价永远比宣传价贵：** 包不全，超出预算，后期还加钱
6. **工程质量差：** 项目经理下拨价钱不高，材料做的也不怎么样

### 低价切入的识别
- 合同价明显低于其他公司 → 一定有大量漏项
- 备注里全是"超过X加钱" → 低开高走
- 水电不估价 → 后期做多少算多少
- 防水只有单价 → 进场坐地起价
- 基层处理不含贴布挂网 → 后期加钱

## 六、只问装修一平方多钱的误区

**只问装修一平方多钱，一定小心了。** 这种问法本身就是错误的。

1. **半包全包内容不同：** 同样叫半包，每家包含的项目不同，单价没有可比性
2. **材料档次不同：** 同样是乳胶漆，100元一桶和2000元一桶差20倍
3. **工艺做法不同：** 同样是贴砖，水泥砂浆铺贴和瓷砖胶薄贴价格不同
4. **工程量不同：** 单价高但面积算得对 vs 单价低但面积故意少算后期加

**正确做法：** 不要只问单价，要看详单式预算，逐项比较。看项目是否齐全、做法是否合理、材料品牌型号是否写清。

## 七、预算过不了以下10关，谁敢急着装修

1. 量房是否准确
2. 项目是否齐全（有没有漏项）
3. 做法是否明确（备注有没有加钱条款）
4. 材料品牌型号是否写清
5. 基层处理是否包含所有工序
6. 防水做法是否规范、面积是否算够
7. 水电是否估价谈实际出入
8. 开关插座安装是否包含
9. 倒角、包管道、过门石是否包含
10. 垃圾清运、成品保护、管理费是否合理
