# 避坑选择找商家类

> 本文件提炼自「从学到装｜装修课堂」知识库15篇全屋定制避坑原创文章，涵盖找商家诀窍、套餐陷阱识别、计费方式选择、电器协调、审美法则、环保避坑等核心内容。

---

## 一、找商家的核心诀窍：人品+经验+审美

全屋定制找法和找装修机构一模一样，最重要的考察对象是人，而非品牌。

**人品是门槛。** 人品端正的商家光明正大赚钱，不会在板材上动手脚，不会用术语糊弄你，不会只图赚钱不管家人健康。判断商家人品从以下几点立马能看出来：

1. **不说E0级谎言。** 国家标准GB 18580-2017已取消E0级，只有E1级。还在用"E0级"忽悠客户的，严重专业失误，纯粹只图赚钱。人造板没有百分百环保的，这才是真相。
2. **不吹颗粒板比实木好。** "大家都在用"是心理暗示，随大流吃亏不寂寞。赚你的钱要你全家的命，你还帮着数钱。
3. **必问入住人口。** 不问家里入住什么人就信口开河大量做人造板的，连家人健康都不考虑，再大品牌也没用。
4. **不使劲给儿童房推人造板。** 孩子刚分房单独住，使劲给儿童房做颗粒板的，可以说是只图赚钱不顾健康。
5. **不是二道贩子。** 没有自己厂子的定制商尽量不找，中间加点钱给你做而已，人员工资、店面营销费用都得加到中间环节。
6. **全屋统一定制。** 只做衣柜不做橱柜和浴柜的商家不找，要不就是二道贩子，要不就是只做赚钱的。

**经验是基础。** 好的设计靠经验积累，没有经年经验，充其量是定制盒子。设计是全屋定制的灵魂，含金量最高。好的设计者自身有很高审美，能引导业主做选择。

**审美是底线。** 全屋定制如果不懂审美，一旦高调就输了。专业做设计的人比你见过的多，什么好看什么不好看，百分之九十是对的。好的定制商一定要有审美，否则这么丑你能过关，说明你没有把握住而已。

## 二、必须淘汰的7类商家

遇到以下情况，直接淘汰，不用犹豫：

1. **满嘴百分百环保的**——人造板不可能百分百环保，说这话的是明着骗你。
2. **红口白牙说颗粒板比实木好的**——只图赚钱不顾健康。
3. **不问入住人口就大量推人造板的**——间接害命。
4. **用大品牌做噱头的**——全屋定制看品牌本身就是误区，品牌不等于板材好，90%的板材品牌自己不生产板材，都是代加工贴牌。
5. **让你糊涂交大定的**——动不动几万定金，后期设计不断加钱，定金不可能退。
6. **只会做衣柜不做橱柜的**——不是全屋定制，是二道贩子。
7. **不考虑见光板、衣柜不通顶、吊柜不挨吊顶的**——只会做盒子，不会设计。

## 三、套餐与低价的陷阱

### 套餐必坑
和装修套餐一丘之貉，坑不死你绝对不可能。全屋定制是非常个性的定制活动，哪有什么标准？套餐的实质就是——专门哄骗爱占小便宜、梦想天上掉馅饼的客户。

- 你看到的低价是按照最简单、最不符合你家习惯的设计算的
- 后期任何个性化需求都要加钱：要抽屉加钱、要层板加钱、要套色加钱、要做满加钱
- "不限平方""两房两厅全屋定制"这种套餐，到最后三万都包不住

### 低价坑
你家懂行的一看需要两万，故意给你设计成一万出头，让你交大定。设计的时候你这里要个抽屉，那里加个层板，三四万就有了。你以为一万搞定的可能掉进三四万的坑里。

### 低价从板材质量下手
逼着商家低价，原来100元的板材换成80元的，做成安装后全部封边处理，你无法判断是否掉包。板材的好坏唯有把不同档次的放一起一目了然。可以让商家让你带走一块样品，安装到家时和打孔的横截面进行对比。

### 免费赠送的陷阱
动不动送你木门、送你一件定制家具的，你以为你长得好看？不是，因为你长得好坑。天下没有免费的午餐，接受了免费的东西，后面必然出事。

## 四、计费方式：投影计价完胜展开计价

### 一定要找投影计价的商家
投影计价最好，不论你有多么个性的要求，总价不会有太大变化。展开计费则你越有要求，价位水涨船高。

### 投影计价必须问清的10个问题
1. 抽屉是否单独收费
2. 层板多少是否单独收费
3. 套色是否单独收费
4. 个性化格局是否单独收费
5. 五金是否单独收费，什么五金需要自己购买
6. 板材厚度多少
7. 背板厚度多少
8. 见光板是否单独计费
9. 安装是否单独计费
10. 上楼是否单独收费

### 好的定制应该是
货真价实，多钱就是多钱。做的层板多少和抽屉多少不会有任何费用，不会一门心思就想让你上套然后不断加钱。如果抽屉、层板、边线都要单独收费，基本可以断定你找的是二道贩子，绝对不是厂家。

## 五、全屋统一定制：千万别分多家

**所有定制一定要找一家去做，千万不要衣柜一家、其他一家，这个完全是自己坑自己的行为。**

原因：
1. **颜色不统一**——不同厂家颜色不可能一样，全屋定制需要整体设计板材颜色搭配
2. **协调出错率高**——工程协调商家越多，出错的几率越多
3. **售后麻烦**——后期颜色以及售后都非常麻烦，风险更大
4. **与装修基础严密衔接**——全屋定制和装修基础以及水电改造严密衔接，一旦出错更麻烦

**装修的可以做全屋定制更好**，设计师是一个人，更好协调，出错几率降低，同时可以压低成本。但全屋定制绝对做不了装修，千万不要找做全屋定制的做装修，相当于找了一个卖刀的给你家做厨师。

## 六、全盘考虑与电器协调

全屋定制并不是独立存在的，做装修方案时一定要全盘考虑、整体规划。即使单独找全屋定制，也必须让做装修的给你规划位置以及最起码的做法。

### 常见脱节翻车案例
- **书桌与衣柜冲突**：前期不考虑，后期只能成品放置，导致衣柜门无法开启
- **开关面板位置**：不提前规划，衣柜缩小一大截空间浪费就放个开关
- **空调洞位置**：不管不顾导致衣柜门打孔，冷凝水管经过衣柜有渗漏风险
- **衣柜与床头柜冲突**：不断开设计导致每次开门都要挪动床头柜

### 电器协调避坑要点
1. **洗碗机**：先让设计师看是否满足空间，水电交底定制必须在场。给定制洗碗机的详细尺寸或直接给网购链接，让定制去协调。不要给文字性尺寸，要有产品图片的尺寸。
2. **零嵌冰箱**：给了产品尺寸结果做成嵌不进去的，定制死不承认。所以千万别图优惠便宜，要的优惠越多离坑越近。
3. **烤箱**：电器销售人员给的尺寸也可能失误（没量到把手凸出部分）。实体店购买一定要有产品本身的图片尺寸以及线上链接尺寸，千万不要让销售人员随便给尺寸。
4. **烟机**：留空小了压根进不去。解决办法：重新改柜子成本高，或烟机上方柜门重新做大，烟机不嵌入完全在吊柜下方。
5. **扫地机器人**：装修前就要规划好尺寸大小位置，定制时根据机器人大小定制。
6. **洗碗机把手**：建议不要带突出把手，内嵌隐形把手是最佳选择，不会与旁边抽屉柜门冲突。

**核心原则**：业主千万不要直接给定制商纯数据，网购直接给链接有详细尺寸。最简单的办法是直接给产品链接和型号，让定制设计师与电器商家沟通，后期错误定制商承担。

## 七、审美避坑9大法则

全屋定制设计和装修设计一样，甚至比装修设计更复杂，并不是定制盒子那么简单。牢记以下9个不做：

1. **不找套餐，不找所谓百姓品牌**——套餐必坑，个性需求越多越贵
2. **没有审美不做乱色**——不套色一个颜色就行，套色需要美感、比例、技巧
3. **不要高调张扬**——弱化存在感是基本常识，丑的地方尽量遮盖隐藏
4. **最好不用亮光材质**——亮光材质自带廉价感
5. **能不用把手不用把手**——公共区域绝对不能用把手，首选免拉手拉槽设计
6. **不能做推拉门**——不仅占用柜内空间，而且自带廉价感
7. **不做大面积玻璃柜门**——不仅贵而且难看凌乱，是展示用的不是收纳用的
8. **不盲目做一门到顶**——必须装修基础横平竖直，板材高于两米四造价高变形几率大
9. **满嘴百分百环保的直接淘汰**——人造板不可能百分百环保

### 颜色选择原则
- 颜色越浅越好，深色必然压抑（除非卧室超级大）
- 造型越少越好，把手极简最好
- 能镶嵌尽量镶嵌，否则见光板比石膏板贵多了
- 即使不镶嵌也要做上三边边线，视觉上不会显得廉价感

### 定制翻车的两大原因
1. **找对定制商**——有设计要求尽量不要找所谓大品牌的百姓品牌定制，那就是批量工业产品，和设计没有半毛钱关系，充其量就是生产盒子。一定要找有设计感的、对设计负责的定制商。
2. **业主坚持己见**——大部分外表已经够好看了，但业主非要这里加个层板、这里要个镂空、这里必须装上把手，全毁掉了。毕竟业主说了算。

## 八、环保避坑核心

### 人造板不可能百分百环保
国家标准E1是单位面积的污染对你伤害不大，并不是没有污染。你把几十平方的人造板堆积到一块，绝对不环保。用人造板谈环保本身就是自欺欺人。

### 总量才是判断标准
- 污染看总量，不是一张环保证书
- 环保证书和学历一个道理，有教师证不代表此人为人师表
- 人造板的环保在什么时候绝对成立？只用半个平方的时候

### 根据入住人口选择板材
1. **婚房不要小孩**——可以随意，无所谓污染
2. **婚房一两年要小孩**——千万谨慎，控制污染总量
3. **有孩子10岁以下**——不能随心所欲，控制污染
4. **孩子未分房**——主卧不要使劲用颗粒板
5. **孩子刚分房**——儿童房不要做颗粒板

### 务实做法
不要因噎废食，不看经济情况一听污染就全部实木。孩子房间或孩子和大人一块住的房间做实木即可，其他人造板就行，既保证基本健康又节省费用。

## 九、定金与报价避坑

### 定金坑
- 全屋定制必须交定金才能上门测量，但一定搞清楚计费方式才能交
- 最好用你家户型图让直接估价，能承受价格、搞清楚板材五金后再交定金上门量房
- 动不动让你交两三万定金的，压根不可能退
- 测量完毕给精确报价后，再交预付款开始具体设计

### 笼统报价坑
- 进店笼统报价一定问清到底实际设计出来能差多少
- 如果是展开计费，你的要求越多总价越高
- 一定是投影计价，而且抽屉层板套色边线随便做，总价不变

### 做定制前要门当户对
- 不要有钱选个低价，也不要没钱想着实木
- 投影面积只能接受一千以内的话，不要想爱格板和激光封边
- 如果这个价格是爱格板，百分百假货
- 低价买好货都让你碰到了，哪有那么好的事情

## 十、好商家的标准

### 好的定制标准
1. **环保**——根据入住人口搭配板材，而非信口雌黄只赚钱
2. **设计**——外表颜值以及功能的合理化
3. **以人为本**——脱离入住人口的定制只是生意

### 好商家的表现
- 货真价实，多钱就是多钱
- 层板和抽屉再多都是你家板子裁割出来的，不会加钱
- 出问题能无条件维修
- 不怕出问题，出问题能承担责任解决问题，这就是商家的本分

### 定制范围参考
全屋定制包括：鞋柜、玄关柜、餐边柜、橱柜、衣柜、衣帽间、浴室柜、洗衣柜、过道镶嵌柜、电视墙镶嵌柜、书柜书架、飘窗柜、卡座等。好的定制商应该能做全部这些，而不是只做其中几样。

## 附：找商家快速自检清单

| 检查项 | 合格商家 | 需淘汰商家 |
|--------|----------|------------|
| 环保说法 | 告诉你人造板不百分百环保，看总量 | 满嘴E0级、百分百环保 |
| 入住人口 | 主动询问，根据人口推荐板材 | 不问不答，一股脑推人造板 |
| 计费方式 | 投影计价，抽屉层板不限制 | 展开计费，每项单独收费 |
| 定金 | 少量定金上门测量，精确报价后再付 | 动不动几万大定，不退 |
| 定制范围 | 全屋统一定制，什么都能做 | 只做衣柜或只做橱柜 |
| 见光板 | 主动考虑，不低价切入 | 不考虑，故意不说 |
| 衣柜做法 | 通顶镶嵌，视觉统一 | 不通顶不镶嵌，做盒子 |
| 吊柜做法 | 挨着吊顶，不留缝隙 | 不挨吊顶，低价糊弄 |
| 审美引导 | 引导低调弱化存在感 | 任由业主高调张扬 |
| 售后 | 出问题无条件维修 | 死不承认，宁愿不要尾款也不改 |
