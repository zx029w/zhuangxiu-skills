# 标准预算结构（半包 / 全包怎么分、怎么算不超）

> 配套「装修预算计算器」Skill 使用。真实装修预算表通常就两块：**基础预算明细（半包）+ 主材明细**——先把这两块分清楚，再算账。

---

## 一、真实预算表结构：基础预算明细 + 主材明细

### 1. 基础预算明细（半包）
- **含**：施工人工 + 辅料（水泥、沙子、腻子、胶、石膏板、龙骨、防水涂料、电线、水管等）+ 水电改造 + 部分现场管理
- **不含**：瓷砖、地板、门、洁具、橱柜、全屋定制、开关灯具、美缝主材等
- **特点**：业主主材自购、价格透明，但需花精力跑建材

### 2. 主材明细
- **含**：瓷砖、地板、木门、钛镁合金门、洁具、橱柜、全屋定制（衣柜/鞋柜/餐边柜等）、开关灯具、美缝、过门石、窗台石、踢脚线、垭口套等
- **注意**：不同公司/不同预算表口径不同——**有的主材明细里已经包含全屋定制，有的把柜子单独列出来**。算总预算前必须先确认这一点，否则容易重复算或漏算。
- **坑**：套餐式全包常"低价切入"，升级项、漏项多，签前务必看详单

### 3. 间接费 / 管理费
- 现场保护、材料搬运、垃圾清运、管理费（西安常见 10%）等，通常放在基础预算明细里或单独列一项。

## 三、主材明细清单模板（确认是否含全屋定制）

### 基础主材
- [ ] 地砖 800×800（约 130 元/㎡）
- [ ] 墙砖 400×800（约 25 元/块）
- [ ] 实木复合烤漆门（约 1300 元/樘）
- [ ] 钛镁合金门（卫生间约 1300~1700 元/樘）
- [ ] 美缝（约 26~40 元/㎡）
- [ ] 过门石、窗台石、垭口套、踢脚线
- [ ] 开关面板、灯具、地漏、背涂胶、瓷砖胶

### 全屋定制（可能在主材明细内，也可能单列）
- **不按死面积算**。虽然参考价写 900 元/㎡ 投影，但每家柜子数量差异很大，按面积乘出来的数往往不准。
- **按经验区间估**（更贴近真实，别死套 900×面积；详见 `西安参考价目表.md` 第四节）：
  - 80㎡（两室/小三室）：**1.5~2.5 万**
  - 90㎡ 三室一卫：**2~3 万**
  - 100㎡ 三室：**2.5~3.5 万**
  - 120㎡ 三室两厅两卫：**3~4 万**
  - 150㎡ 四室：**4~6 万**
  - 旧房沿用老柜可低至 1~1.5 万；满墙柜/衣帽间可超 6 万；具体看衣柜、鞋柜、餐边柜、橱柜各做多少
- 如果预算表里柜子是单独一项，直接看那一项金额；如果含在主材里，要先确认清楚，再算总账。
- 见光板（300 元/㎡）、玻璃柜门（+500 元/㎡）、特殊五金另计。

> 如果装修公司给你的主材明细里已经含全屋定制，总预算直接加两块；如果主材不含柜子，柜子要单独加一列，按经验区间估，别漏也别重复。

## 四、不超预算实操（业主照做）

1. **先确认预算表结构**：拿到报价单先看是「基础预算明细 + 主材明细」两块，还是主材里已经把全屋定制单独拆出来。确认口径前不算总账。
2. **先定总价上限**：根据家庭存款和贷款能力，定一个"死线"，所有分配围着它转。
3. **按基础+主材两块分配**：基础预算明细填施工辅料，主材明细填砖/门/洁具/柜子，两块相加再加管理费就是总预算。
4. **主材自购留 15% 机动**：砖、门、洁具、定制柜实际常超参考价，预留缓冲。
5. **水电留弹性**：估价只是参考，完工按实决算，合同里写清"估价 ±X% 内不补差"。
6. **定制柜盯见光板 / 五金**：报价单常不含见光板、拉篮、反弹器，签前问清。
7. **管理费比例写死**：10% 是西安常见值，超过要问为什么。
8. **不签"一口价死合同"**：装修变数多，死合同要么偷工减料，要么增项扯皮。

## 五、预算最容易超的 6 个点（⚠️ 重点提醒）

1. **水电按实决算**：前期估价低，完工量上来超 30% 很常见。
2. **主材明细口径不清**：没搞清"含不含全屋定制"，要么重复算，要么漏算一大块。
3. **定制柜漏项**：见光板、五金、特殊拉篮不写进预算，后期全是增项。
4. **主材自购漏算**：只算了砖，忘了门、垭口、踢脚线、美缝、过门石。
5. **旧房拆除费**：新房没有，旧房拆改 + 铲素灰 + 拆保温，轻松多花大几千。
6. **管理费 / 设计费**：小项加起来占比不低，签前确认比例。
