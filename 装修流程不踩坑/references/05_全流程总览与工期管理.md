# 全流程总览与工期管理

## 一、全流程总览

### 1. 装修的完整流程
收房验房→确定装修方式（半包/全包/清包）→找装修机构→量房设计→出方案报价→签合同→办理装修许可→开工交底→拆除砌墙→水电改造→瓦工（防水/贴砖）→木工（吊顶/造型）→油漆（墙面施工）→安装阶段（橱柜/门/开关/灯具/洁具/地板）→开荒保洁→家具家电进场→软装配饰→通风晾房→入住

### 2. 装修前必须想清楚三件事
**① 你家到底要怎么装修？**
- 不要张口问设计师"你给我看着做"
- 先想清楚：住几口人？地面铺瓷砖还是木地板？墙面乳胶漆还是壁纸？要不要吊顶？要不要背景造型？
- 例如60平的房子，半包两三万可以装，七八万也可以装，要求变化预算天差地别

**② 你的预算能否支撑你找装修机构？**
- 自己装修还是找装修公司，由你的钱包决定
- 对细节风格要求高→必须找装修机构→花钱必然多
- 有游击队的价格+装修公司的服务？不存在的

**③ 装修流程怎么落地**
- 找装修公司：每一步人家提前会跟你说，你无需操心
- 找散工/施工队：只能自己操心，你没有管控大局的能力就别找

### 3. 半包vs全包vs清包
- **半包**：装修公司负责施工+辅材，业主买主材（瓷砖、地板、洁具等）
- **全包**：装修公司负责全部，业主拎包入住
- **清包**：业主自己买所有材料，只找工人干活（最累、最适合有经验的人）
- 半包全包只是名字，关键看每家内容不一样，要问清具体包含什么

## 二、工期管理

### 1. 100平房子装修工期参考（一天不休息）
以100平小三室为例，从签订合同起：

| 阶段 | 时间 |
|------|------|
| 拆除→水电完工 | 约7天 |
| 水电完→闭水试验 | 约7天 |
| 瓦工（贴砖） | 约20天（含养护） |
| 木工+油漆工 | 约25天 |
| 瓦工完工→量取木门 | 40天内等木门到货 |
| 木门+橱柜安装 | 约3天 |
| 开关插座灯具安装+修补 | 约7天 |
| **总计（不含节假日）** | **约70天** |

注意：
- 如果周末和节假日不让施工，100平房子签合同至少3个月才能按时完工
- 不要相信装修公司号称20天完工的谎话
- 设计不算在工期内，从签合同之日起算

### 2. 催工期的风险
- 催工期最容易出质量问题
- 每一道工序必须干透才能做下一步：
  - 腻子没干透就刷漆→后期开裂起皮
  - 防水没干透就贴砖→防水层被破坏
  - 瓷砖没干透就踩踏→空鼓
- 装修合同虽然有工期约定，但赶出来的活质量堪忧

### 3. 工期延期的常见原因
- 节假日不让施工（周末、法定假日）
- 停水停电
- 业主主材没及时到场（自己买的瓷砖、门等）
- 协调工人时间
- 天气原因（梅雨季潮湿，墙面干不透）
- 装修过程中改了方案加了项目

### 4. 各阶段需要提前下单的东西
- **木门**：瓦工完工后就要量尺下单，生产周期40天左右
- **橱柜**：水电完工后就要量尺设计，生产周期30天左右
- **定制衣柜**：油漆完工后量尺，生产周期30-45天
- **窗户/封阳台**：开工前就要定，生产周期15-20天
- **瓷砖**：开工前定，水电完前要到场
- **地板**：油漆完工前定，安装阶段到场

### 5. 工期管理的核心原则
- 早定晚到：能提前定的东西不要拖
- 交叉安排：不互相影响的工作可以同期进行
- 留有余地：时间表要宽松，给每个阶段留缓冲
- 合同明确：签合同时明确工期和延期违约责任

## 三、全流程时间轴(简化版)

```
第1-3天：拆除铲墙
第4-7天：水电改造(含交底)
第8-12天：做防水+闭水试验
第13-22天：铺贴瓷砖(瓦工)
第23-25天：瓦工养护
第23-30天：木工吊顶(可与瓦工交叉)
—— 瓦工完工同时量门、橱柜尺寸 ——
第30-50天：墙面基层处理+油漆(等木门到货)
第50-55天：木门安装+橱柜安装
第55-60天：开关插座+灯具+洁具安装
第60-65天：地板安装+开荒保洁
第65-70天：家具家电进场+软装
第70+天：通风晾房
```
