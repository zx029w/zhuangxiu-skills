> 本文件收录全屋定制流程规划、精装房微改、定制区域规划等核心知识，来源：从学到装｜装修课堂知识库。

## 一、全屋定制流程

### 1. 标准流程

1. **学习了解** → 先了解全屋定制常识
2. **估价** → 根据户型图尺寸估价
3. **交定金** → 估价能接受后交定金（一两千即可），上门量尺
4. **精确预算** → 量尺回来计算准确预算
5. **付一半费用** → 预算确定后付50%，开始设计施工图和配色图
6. **图纸确认** → 所有图纸确认完毕
7. **付95%款项** → 下单制作
8. **安装完毕** → 付清5%尾款

### 2. 计价方式
- 投影面积计价
- 抽屉层板不限制随便做
- 包含国产五金DTC、普通挂衣杆、普通抽屉滑轨
- 特殊五金提前沟通（业主自购或商家提供）
- 两年售后（非人为破坏免费维修更换）

### 3. 注意事项
- 前期不收取费用（效果图除外）
- 效果图不是必须的：实景图能表现方案就不用做效果图
- 全屋定制不能脱离基础装修，否则无法衔接

## 二、家里哪些地方需要全屋定制

### 12个定制区域

1. **门厅鞋柜** — 到顶定制，可做换鞋凳，成品鞋柜达不到收纳量
2. **餐边柜/卡座** — 满足水吧台功能，弥补厨房收纳不足，可镶嵌冰箱
3. **电视柜** — 刚需房做满墙收纳柜比造型实用；怕压抑可做悬空柜
4. **沙发背景收纳柜** — 移动沙发，沙发背景也能做收纳
5. **过道收纳柜** — 利用过道纵深，与卧室做双面柜
6. **阳台收纳柜** — 洗衣柜+吊柜+清洁柜
7. **卧室衣柜** — 侧墙衣柜或床尾衣柜（可兼顾书桌/化妆台）
8. **书桌/化妆台** — 衣柜延伸书桌，兼顾书桌+化妆台+床头柜
9. **榻榻米** — 收纳神器，刚需房同时拥有卧室和书房功能
10. **书房书柜** — 小空间高收纳必须定制
11. **橱柜** — 属于全屋定制项目
12. **浴柜** — 根据空间定制比成品更实惠

### 铁律：全房定制一家做完
橱柜和衣柜必须找同一家做：
- 颜色款式设计统一协调
- 后期出问题不用多家扯皮
- 分开做每家都要利润，费钱不讨好
- 只会做橱柜或只会做衣柜的，百分百不是自己做的

## 三、精装修房定制窍门

精装修不可能大面积拆除，定制需要"动脑子"解决冲突：

### 1. 吊顶/灯带冲突
- 吊顶太窄做不了通顶柜 → 做悬空电视柜/不通顶柜
- 上方有灯带 → 不通顶，结合悬空做法
- 中央空调风口冲突 → 采用悬空做法避开风口

### 2. 石膏线冲突
- 没有吊顶只有石膏线 → 裁割石膏线实现定制
- 瑕不掩瑜，美观和功能说得过去

### 3. 空间处理技巧
- 吊顶过窄：做矮柜与冰箱放置，不压抑
- 吊顶太窄：上下悬空定制，保留吊顶和灯带
- 阳台小：加宽吊顶或木质饰面延伸，成就阳台柜
- 过道长：不做通顶鞋柜，矮柜加长减少压抑感
- 高度太高无吊顶：白色浅色系定制，下方木色减少压抑

### 4. 精装修定制原则
- 不可能完美，只能根据现有格局尽量满足
- 精装修定制永远没有毛坯房定制效果好
- 好的定制商能用经验弥补精装修的缺陷

## 四、该不该一次性做好所有柜子

### 情况一：经济紧张，入住人少
- 可以不做其他卧室衣柜，空置
- 只满足1-2人基本生活刚需
- 优点：缓解当下经济紧张
- 缺点：分期花钱，总体不少花，后期入住折腾

### 情况二：婚房准备生小孩
- 次卧可以空置（孩子小和父母住）
- 儿童房需要此时定夺：
  - **现在做人造板**：造价低 + 有放置期 + 孩子分房时刚好达标
  - **以后做实木**：缓解当下经济 + 但孩子分房时必须买实木（造价高）
- 两种各有利弊，关键看经济状况

### 一次性做的好处
- 整体风格颜色统一
- 一次性整体造价好谈
- 分开多次每次数量少，造价无形中变高

### 分期做的前提
- 经济紧张
- 入住人少
- 后期能接受折腾

## 五、用定制隐藏管道（红砖包管道多此一举）

### 厨卫管道（必须红砖包）
- 有水有潮气
- 高层需要隔音
- 墙面需要铺贴瓷砖
- 红砖包裹后抹灰贴砖是常规做法

### 阳台管道（不需要红砖包）

**情况一：阳台与客厅打通，不贴砖**
- 直接定制收纳柜隐藏管道
- 管道前面还可以满足镂空收纳和展示
- 经济好看，不占地方

**情况二：阳台做休闲书桌区域**
- 石膏板包裹管道，刷同色乳胶漆
- 利用夹角定制书桌/书架
- 管道隐藏于无形

**情况三：有中央空调，扩充阳台空间**
- 管道用石膏板隐藏装饰
- 刷同色乳胶漆
- 轻巧不占地

**情况四：精装修阳台（真石漆墙面）**
- 不放洗衣机时：麻绳缠绕装饰
- 放洗衣机时：定制高柜隐藏管道 + 清洁柜收纳

### 核心原则
设计之初就规划好定制方案，管道无需单独掏钱包裹，后期直接用定制隐藏——省钱、好看、省空间。
