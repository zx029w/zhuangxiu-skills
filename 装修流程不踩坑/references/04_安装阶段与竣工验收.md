# 安装阶段与竣工验收

## 一、安装阶段

### 1. 安装顺序（非常重要！）
油漆完工→橱柜安装→油烟机灶具安装→木门安装→踢脚线安装→开关插座安装→灯具安装→洁具五金安装→地板安装→定制柜安装→开荒保洁→家具进场→软装配饰→通风晾房

注意：
- 地板最后装（避免其他安装时刮花地板）
- 橱柜先装（方便后续水电收口）
- 木门定制周期最长（40天左右），要提前下单

### 2. 开关插座安装要点
- 开关插座的高度和位置在瓦工阶段已经预留好，安装阶段只是接上线装上
- 开关距地1.3-1.4米（方便使用）
- 普通插座距地0.3米
- 厨房台面插座距地1.2-1.3米
- 卫生间插座距地1.4-1.5米（远离淋浴区）
- 空调插座距地1.8-2米
- 每个房间开关装在进门侧

### 3. 开关插座数量参考
- **主卧**：床头两侧各1个插座+1个开关、空调1个、电视1个、预留1-2个
- **客厅**：电视墙3-4个、沙发两侧各1-2个、空调1个、预留1个
- **厨房**：台面4个带开关、冰箱1个、油烟机1个、水槽下2个（净水器/垃圾处理器）
- **卫生间**：镜柜旁1-2个、马桶旁1个（智能马桶）、热水器1个
- **餐厅**：餐桌旁1-2个（火锅/手机充电）

### 4. 厨房安装费用参考
一个普通厨房安装：
- 橱柜：500-1000元/延米（含台面）
- 油烟机+灶具：3000-8000元
- 水槽+龙头：500-2000元
- 厨房电器：根据品牌差异大

### 5. 卫生间安装费用参考
一间普通卫生间安装：
- 马桶：1000-5000元
- 浴室柜+镜柜：1500-5000元
- 花洒：500-3000元
- 热水器：1500-4000元
- 浴霸：500-1500元
- 淋浴房/浴帘：500-3000元

### 6. 安装阶段注意事项
- 安装好每一件都要测试是否正常工作
- 橱柜安装完检查柜门开合是否顺畅、抽屉推拉是否顺滑
- 木门安装完检查门缝是否均匀、门锁是否好用
- 灯具安装完检查是否稳固、亮度是否足够
- 洁具安装完试水（检查上下水是否通畅、有无渗漏）
- 地板安装完踩一圈检查是否有异响

## 二、竣工验收

### 1. 竣工验收检查清单

**① 水电方面（安全隐患，必须检查）**
- 每个开关都试一遍：是否有接触不良、打火
- 总开关试一下：灵敏度
- 每个插座都插电测试：用充电器插头逐一检查接头是否松动、是否通电
- 洗手盆、水槽、淋浴都试一遍水路：检查下水是否通畅
- 低头检查下水管、角阀、软管：是否有漏水渗水
- 检查强弱电箱：标识是否清楚、空开是否正常

**② 墙面和吊顶**
- 石膏板接缝处：注意观察是否开裂（热胀冷缩产生裂缝）
- 所有墙面：仔细观察是否有裂缝
- 厨卫外面的乳胶漆墙面：重点观察是否有裂缝或鼓包（潮气导致）
- 发现问题及时维修，入住后麻烦

**③ 缝隙和收口**
- 踢脚线与墙体之间玻璃胶：是否开裂
- 木门与墙体之间玻璃胶：是否开裂
- 厨卫美缝：是否完整无开裂
- 所有打胶的地方在干燥后可能开裂，需要补胶

**④ 定制木作**
- 橱柜柜门：开合是否正常
- 衣柜柜门：推拉或平开是否自如
- 所有柜体：是否有变形、磕碰
- 抽屉滑轨：是否顺滑

**⑤ 门窗**
- 窗户开关是否顺畅
- 密封条是否完好
- 纱窗是否完整
- 门锁是否好开关

### 2. 通风晾房注意事项
- 晾房期间出门切记关闭电闸和水路总阀（安全第一）
- 装修完毕后需要通风晾房一段时间
- 晾房期间多去检查，发现小问题及时让装修公司维修
- 墙面开裂、玻璃胶开裂等小问题在晾房期间发现最好
- **装修污染超标的话，单纯晾房意义不大**——甲醛挥发周期长达数年，最有效的是通风+源头治理（用环保材料）

### 3. 软装进场
- 基础硬装全部完成后才开始软装
- 家具尺寸要提前量好（特别是定制家具）
- 大件家具先入场，小件软配后入场
- 软装搭配不要盲目跟风，以实用为主

### 4. 入住前准备
- 开荒保洁（装修垃圾清理、全屋清洁）
- 检查所有家电是否正常
- 检查所有水电位是否正常
- 办理入住手续（物业登记）
- 搬家时注意保护墙面和地面
