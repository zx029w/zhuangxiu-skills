# 木工与油漆阶段

## 一、木工阶段

### 1. 木工施工顺序（与瓦工的先后关系）
- 装修大顺序：水电→瓦工→木工→油漆→安装
- 常规做法：先铺完瓷砖（地面凝固保护），再做石膏板吊顶
- 但有些人担心铺完瓷砖木工施工会损坏瓷砖，想先做木工再铺砖——这很难实现
- 除非你家地面没有任何瓷砖铺贴，或者全屋通铺木地板（先做吊顶后找平）

### 2. 吊顶施工要点
- 石膏板吊顶用轻钢龙骨（比木龙骨更牢固、防火）
- 石膏板接缝处要做倒V形处理（方便嵌缝）
- 石膏板转角处要用整板裁切（L形），避免后期开裂
- 石膏板与墙面之间留缝3-5mm（防开裂）
- 吊顶内管线要提前布好（灯线、新风管道、空调管道）

### 3. 厨卫吊顶
- 厨卫有大梁时，吊顶要包梁处理
- 铝扣板吊顶更容易检修（拆卸方便）
- 石膏板吊顶好看但检修麻烦，厨卫建议用铝扣板
- 吊顶高度一般2.4-2.5米

### 4. 木工验收要点
- 检查吊顶水平度（用水平仪）
- 检查石膏板接缝处理是否规范
- 检查灯位预留是否正确
- 轻敲石膏板看是否固定牢固
- 检查石膏板表面是否有破损

## 二、油漆阶段（墙面施工）

### 1. 墙面施工为什么容易出问题
90%的乳胶漆翻车都卡在两个问题上：
- 墙面反复开裂、起皮
- 墙面返碱、泛白、发霉

### 2. 墙面开裂的根源——基层处理不到位
墙面开裂的根不在漆，在基层！10道正确做法：

**① 素灰铲除**：新房毛坯房大部分有素灰，必须铲除到底露出原始基层。手工铲不干净要用角磨机打磨。

**② 大白铲除**：开发商刮的大白必须铲干净，后期刷墙固批腻子。铲不干净会导致后期开裂脱皮。

**③ 松散沙灰层**：混凝土墙体一定要露出混凝土本来面目。沙灰层如果掉渣掉粉，必须铲除到底。

**④ 老房铲除**：老房水泥层检查空鼓，有空鼓敲掉；没有空鼓且坚硬则无需铲到红砖层。

**⑤ 沙灰层松散**：新房非承重墙用坚硬物品划一下，掉粉脱落要铲除到底。

**⑥ 墙面空鼓**：装修前敲打一遍，空鼓必须敲掉铲除，重新粉刷石膏找平。不要用水泥砂浆，容易脱壳。

**⑦ 墙面挂网**：良心油漆工会做的。旧房铲墙99%需要全房挂网。新房大面积粉刷石膏找平也要挂网。沙灰层松散也需要挂网。

**⑧ 不同材质墙体缝隙**：最低标准是一道牛皮纸+两道网，否则必裂。

**⑨ 石膏板缝隙处理**：同样一道牛皮纸两道网。

**⑩ 石膏砂浆抹灰层**：必须全房敲空鼓，否则后期必然大面积脱壳。

### 3. 乳胶漆施工要点
- 乳胶漆建议：**顶面喷漆、墙面刷漆**
  - 顶面喷：速度快、均匀、喷不到的地方少
  - 墙面刷：滚刷更有质感，后期修补色差小
- 墙面刷漆至少一底两面（一遍底漆两遍面漆）
- 每遍漆间隔时间：至少4小时（视天气和通风情况）
- 腻子批刮：至少两遍，每遍都要干透才能批下一遍
- 墙面打磨：腻子干透后打磨平整
- 底漆的作用：封闭基层、增加面漆附着力、节省面漆

### 4. 墙面返碱的原因
- 卫生间防水没做好，潮气渗透导致外墙返碱
- 北方（西安）墙面返碱几乎都是渗水或漏水导致
- 预防：卫生间防水做高（至少1.8米，2.4米更好）
- 常年密闭不通风也会导致墙角发霉

### 5. 油漆验收要点
- 检查墙面是否平整（用靠尺或用手电筒侧光看）
- 检查阴阳角是否垂直
- 检查乳胶漆颜色是否均匀、有无色差
- 检查墙面是否有砂眼、流坠、刷痕
- 检查墙角交接处是否笔直
- 检查所有接缝处是否处理到位

### 6. 油漆阶段时间参考
- 100平房子墙面施工：好的油漆工约20天（含批腻子、打磨、刷漆）
- 每道工序必须干透才能做下一步，催工期最容易出问题
