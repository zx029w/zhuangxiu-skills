## 一、负面信息搜索方法（核心功能）

### 搜索平台

1. **搜索引擎**（百度、谷歌、搜狗）
2. **社交媒体**（小红书、抖音、微博）
3. **投诉平台**（黑猫投诉、12315平台）
4. **AI工具**（命令AI帮你收集信息）

### 搜索关键词模板

> 把"XX装饰"替换为你要查的公司名称

**基础关键词（5个组合）：**

1. `XX装饰 垃圾`
2. `XX装饰 坑人`
3. `XX装饰 欺诈`
4. `XX装饰 投诉`
5. `XX装饰 黑`

**进阶关键词（更精准）：**

6. `XX装饰 跑路`
7. `XX装饰 增项`
8. `XX装饰 质量问题`
9. `XX装饰 售后`
10. `XX装饰 避雷`

### AI辅助搜索命令模板

> 可以用以下命令让AI帮你收集信息（来源：2025-04-01、2025-03-30文章）

**第一步：搜索负面信息**
别急着找"哪家好"，先看看"谁最烂"！用AI搜索装修公司的负面信息。

```
请列举 [城市名] 装修公司所有负面信息，
关键词包含：垃圾、欺诈、黑、骗子、坑、跑路、增项、质量、售后、避雷
只列举标题，加上引用链接即可。
```

**第二步：精准定位问题公司**
如果想查某一家公司，比如"XX装饰"，可以修改提示词：

```
请列举 [城市名] XX装饰所有负面信息，
关键词包含：垃圾、欺诈、黑、骗子、坑等，
只列举标题，加上引用链接即可。
```

打开链接，具体问题一清二楚！这样能快速找到全网的负面信息，有链接可查，事实依据更可靠！

**第三步：多换几个AI工具和搜索引擎**
- 多换几个搜索引擎（百度、360、搜狗、谷歌），以防有些实力大的采用公关手段屏蔽掉所有负面信息
- 多换几个AI工具（DeepSeek、文心一言、通义千问等），不同AI搜索结果可能有差异

**注意事项：**
- AI搜索的结果一定要有事实依据的负面信息，不是靠一句诋毁就能判断好坏的
- 通过前两步，能淘汰掉60%的装修公司！剩下的公司，再结合实地考察

### 判断标准

**负面信息多而集中 → 直接淘汰，不用浪费时间**

- 如果搜索结果前3页都是负面信息 → 坚决不选 ❌
- 如果负面信息很少，且有正面信息穿插 → 可以进一步考察 ⚠️
- 如果几乎找不到负面信息 → 可以重点考察 ✅

**注意：不要只看数量，要看质量**
- 是个别业主无理取闹，还是普遍问题？
- 是偶然事件，还是系统性问题？

## 三、其他公开查询渠道

### 1. 企业信用信息查询

- **天眼查** / **企查查** — 查看公司注册资本、成立时间、法律诉讼、行政处罚
- **国家企业信用信息公示系统** — 官方渠道，查看经营异常、严重违法失信名单

### 2. 投诉平台查询

- **黑猫投诉** — 搜索公司名称，查看投诉内容及处理情况
- **12315平台** — 查看投诉记录

### 3. 社交媒体搜索

- **小红书** — 搜索"XX装饰 避雷"
- **抖音** — 搜索"XX装饰 坑"
- **微博** — 搜索"XX装饰 投诉"

### 4. 本地论坛/社群

- 本地装修论坛
- 小区业主群
- 本地生活类APP（如大众点评）
